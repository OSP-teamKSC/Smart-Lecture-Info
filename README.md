# 크누버스(knubus)
- 크누버스(knubus)는 knu + syllabus의 합성어로 기존의 경북대 강의 계획서의 불편한 점 개선
## 개발환경
- python @3.8.10
- mysql @8.0.29
- flask server
## 환경설정
```
# 가상환경 생성
$ python -m venv [your_virtual_environment]
$ source [your_virtual_environment]/bin/activate

# requirements install
$ pip install -r requirements.txt
```
